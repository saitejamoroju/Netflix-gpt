FireBase setup
    - open firebase go to console and create new project 
    - meanwhile install firebase (npm i firebase)
    - after creating project click on web host 
    - copy the Auth code 
     - enter firebase login
    - install dependecies again (npm i firebase-tools)
    - firebase init 
    - select hosting- static web pages
        - do you want to use public directory -no
        - do you want to change file to index.html - no
    - npm build 
    - firebase deploy
